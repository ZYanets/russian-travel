# Проектная работа Яндекс.Практикум: Путешествие по России

## Описание
Одностраничный сайт на тему "Путешествия по России", адаптированный под различные устройства шириной от 320 пикселей.

## Использованные технологии
* Флекс-бокс верстка
* Grid Layout
* Медиазапросы
* Методология БЭМ.
